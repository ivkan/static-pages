# SEO Weekly Reports - Инструкция

Эта папка содержит еженедельные SEO отчёты для **immerss.live**.

## Структура файлов

```
reports/
├── README.md                 # Эта инструкция
├── glossary.html            # Глоссарий SEO-показателей (русский)
└── seo-2025-12-15.html      # Недельный отчёт (пример)
```

## Как создать новый недельный отчёт

### 1. Подготовка данных

#### Google Search Console
1. Откройте [Google Search Console](https://search.google.com/search-console)
2. Выберите свойство `immerss.live`
3. Перейдите в раздел **Performance** (Эффективность)
4. Настройте период:
   - **Дата:** Выберите последние 7 дней (например, 16-22 декабря 2025)
   - **Сравнить:** С предыдущими 7 днями (например, 9-15 декабря 2025)
5. Нажмите **Export** → экспортируйте все вкладки в CSV
6. Сохраните папку с экспортом (например, `immerss.live-Performance-on-Search-2025-12-22`)

**Важно:** Убедитесь, что экспортировали данные с точными датами, а не "последние 7 дней" (это обеспечивает консистентность данных).

#### Bing Webmaster Tools
1. Откройте [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. Выберите сайт `immerss.live`
3. Перейдите в **Search Performance**
4. Настройте:
   - **Date Range:** Custom Range
   - **Current Period:** те же даты, что и в Google (16-22 декабря 2025)
   - **Compare to:** предыдущий период (9-15 декабря 2025)
5. Экспортируйте данные в CSV (например, `immerss.live_SearchPerformanceCompare_All_12_22_2025.csv`)

**Примечание:** Bing иногда не показывает данные за последний день — в таком случае используйте период на 1 день короче и в Google тоже.

### 3. Структура данных в экспортах

#### Google Search Console (папка)
```
immerss.live-Performance-on-Search-YYYY-MM-DD/
├── Устройства.csv     # Используем для подсчёта итогов
├── Страны.csv
├── Запросы.csv
├── Страницы.csv
└── Фильтры.csv        # Проверка периода
```

**Нужный файл:** `Устройства.csv` — содержит суммарные данные по всем устройствам (ПК, мобильный, планшет).

#### Bing (один CSV файл)
```
immerss.live_SearchPerformanceCompare_All_MM_DD_YYYY.csv
```

Содержит данные по дням с текущим и предыдущим периодом.

---

### 4. Формат имени файла отчёта

```
seo-YYYY-MM-DD.html
```

Где `YYYY-MM-DD` — **последний день** отчётного периода.

**Пример:**
- Период: 16-22 декабря 2025
- Имя файла: `seo-2025-12-22.html`

## Метрики в отчёте

| Метрика | Описание |
|---------|----------|
| **Clicks** | Количество кликов на сайт из поиска |
| **Impressions** | Количество показов сайта в результатах поиска |
| **CTR** | Click-Through Rate — процент кликов от показов |
| **Avg. Position** | Средняя позиция в результатах поиска (меньше = лучше) |

Подробное описание всех метрик: [glossary.html](glossary.html)

## Частые вопросы

**Q: Какой период выбирать?**
A: Рекомендуется еженедельные отчёты с понедельника по воскресенье (или любой другой фиксированный 7-дневный период).

**Q: Что делать, если Bing не показывает данные за последний день?**
A: Используйте период на 1 день короче в обоих сервисах (например, 9-15 вместо 9-16).

**Q: Нужно ли обновлять glossary.html?**
A: Нет, он универсальный и не зависит от конкретного отчёта.

**Q: Можно ли изменить дизайн отчётов?**
A: Да, можно отредактировать любой существующий отчёт и использовать его как новый шаблон.
